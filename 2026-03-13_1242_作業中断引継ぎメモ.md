## 5. インシデントと対応

### 5-1. 画像にグラデーションが重なって見えた
- 原因:
  - `background-image` ではなく、`.option-card::before` の半透明オーバーレイだった。
- 対応:
  - `option-card-image-right::before { display: none; }` を追加した。

### 5-2. 画像が「右寄せ」ではなく「拡大表示」になっていた
- 原因:
  - `.option-card` の共通 CSS に `background-size: cover;` があり、カード全体に拡大されていた。
- 対応:
  - 右側画像用の共通クラス `option-card-image-right` を作成し、`background-size: auto 100%` にした。

### 5-3. `fragrance-graph.html` の画像外周に余白が見えた
- 原因:
  - 親カードの padding と画像の border-radius。
- 対応:
  - 親カード padding を 0 に変更。
  - 画像 border-radius を 0 に変更。

### 5-4. 文字化けして見える箇所がある
- 状況:
  - PowerShell 上の一部出力で日本語が文字化けして見えた。
  - HTML/CSS 本体はそのまま編集可能だった。
- 対応:
  - ASCII で安定して取れる文脈を基準に `apply_patch` で編集した。
- 注意:
  - 今後も PowerShell 上の表示だけで内容を断定しないこと。
  - ブラウザ表示またはファイル本体の実内容で確認すること。
