# 香りワークショップトップページ レスポンシブ対応

## 対象ファイル

- `香りワークショップ_トップページ.html`（現 `top-page.html`）

## 対応方針

- PC前提で作成していたトップページを、`PC / 中間帯 / スマホ` の3段階で見え方を分岐
- 1280x900 を基準にしたPC完成形を維持しつつ、横幅はあるがPCには少し足りない画面で崩れないように調整
- スマホでは「読める」「押せる」「迷わない」ことを優先して情報量を圧縮
- スマホ幅の最小想定は `375px`

## ブレークポイント

### PC

- `@media (min-width: 68.75em)`  
  目安: `1100px` 以上

### 中間帯

- `@media (min-width: 48em) and (max-width: 68.6875em)`  
  目安: `768px - 1099px`

### スマホ

- `@media (max-width: 47.9375em)`  
  目安: `767px` 以下

## 全体調整

- `min-aspect-ratio: 16/9` によるPC化を廃止
- `section-header` の `max-width` を `1160px` に調整
- アンカー遷移の見え方を維持するため、SPのみ `scroll-padding-top` を拡大

## PC対応

- `hero` と `flow` をPC完成形として維持
- `reassure-grid` を等幅2カラムに変更
- `question-preview` は `preview-pc + preview-balance` の2カラム
- `cta-box` は2カラム構成

## 中間帯対応

### 目的

- PCレイアウトを無理に維持せず、中間幅専用の安定した構成に切り替える

### 主な変更

- `hero` の画面高固定を解除
- `hero-copy` / `hero-visual` の最小高さを圧縮
- `flow-grid` を3列から2列化
- `flow` の3つ目カードは横幅いっぱいに展開
- `flow-visual-card` を非表示
- `benefits-grid` を2列化
- `reassure-grid` を1列化
- `cta-box` を1列化

### Question Preview

- `preview-pc` は中間帯では非表示
- `preview-pad` を追加
- `img/TOP/中間イメージ.png` を使用
- `preview-pad` と `preview-balance` を横並びで表示

### アンカー調整

- ヘッダーナビの `お願い` は `#final-cta-heading` を参照
- 中間帯のみ、クリック時にJSでスクロール位置を補正
- `Final Call To Action` 見出しが上側に収まりやすいようオフセットを調整

## スマホ対応

### ヘッダー

- ハンバーガーメニュー化
- メニューは押し下げではなく、ヘッダー直下に重なるドロップダウン表示
- メニュー項目は横並びチップ表示
- 各メニュー項目は等幅化
- `回答する` CTA はスマホ幅のみ表示
- `予約する` CTA は継続表示

### Hero

- `hero-gallery` を非表示
- `hero-visual` の高さを短縮
- 背景画像は中央表示
- ヒーロー情報量を圧縮して縦長端末でも見やすく調整

### Flow

- `flow-visual-card` を非表示
- `flow-action-card` のみを見せる形に整理
- `3 Step Flow` の横にステップジャンプを追加
- 以下の各カードへスクロールできるように対応
  - `#flow-step-1`
  - `#flow-step-2`
  - `#flow-step-3`

### Question Preview

- `preview-mobile` を使用
- `preview-layout` を1カラム化
- モバイルフレーム幅を調整
- スライダー行の列幅とフォントサイズを圧縮

### Value / Reassurance / Final CTA

- CTAボタン群を全幅化
- `reassurance` 内CTAと上要素の間隔を拡張
- 画像系ブロックの最小高さを圧縮
- `cta-box` の余白を圧縮

## reassurance セクション調整

- 左カラムCTAに `予約する` を追加
- CTAを中央揃えに変更
- 中間帯/スマホではCTA上部に余白を追加
- FAQ文言を圧縮して高さを揃えやすく調整
- PCでは `reassure-grid` を `1fr 1fr` に変更

## Final CTA セクション調整

- セクション見出しを独立した `section-header` として上部に配置
- `Final Call To Action / 私たちからのお願い` に整理
- CTAボタンを横並びに整理
  - `香りづくりをはじめる`
  - `予約する`
  - `上へ戻る`
- ヘッダーナビの `お願い` と連動

## 画像関連調整

- `value-visual-secondary` の背景画像を `img/TOP/Workshop体験.png` に変更
- 同画像の白いグラデーションオーバーレイを除去

## 備考

- トップページのレスポンシブ対応は、この時点で完成扱い
- 今後の修正は微調整か、他ページへの展開を前提とする
